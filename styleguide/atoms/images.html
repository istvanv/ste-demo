Image (and examples for the use of ratio container

To display images

Basic usage

By default, an image component takes up 100% of the parent width.


              <div class="u-mb-x10 u-6/24"><img class="c-image" src="./styleguide/assets/sg-placeholder-material-landscape.jpg"></div>

Sizes (optional)

This sets an initial size as min-width. Note that this overrides the parent width, so will overflow.

Size xl
Size lg
Size md
Size sm
Size xs

              <div class="u-12/24">
                <div class="sg-component-example__comments">Size xl</div><img class="c-image c-image--size_xl" src="./styleguide/assets/sg-placeholder-material-landscape.jpg">
                <div class="sg-component-example__comments">Size lg</div><img class="c-image c-image--size_lg" src="./styleguide/assets/sg-placeholder-material-landscape.jpg">
                <div class="sg-component-example__comments">Size md</div><img class="c-image c-image--size_md" src="./styleguide/assets/sg-placeholder-material-landscape.jpg">
                <div class="sg-component-example__comments">Size sm</div><img class="c-image c-image--size_sm" src="./styleguide/assets/sg-placeholder-material-landscape.jpg">
                <div class="sg-component-example__comments">Size xs</div><img class="c-image c-image--size_xs" src="./styleguide/assets/sg-placeholder-material-landscape.jpg">
              </div>

Constrain display ratio

Any overflow is essentially masked by the container that sets the ratio. Width is set by the parent.

4:3 ratio
16:9 ratio
